2025回顧2026期許

Posted on 2025-12-31 In Life

上個月和朋友聊天時，我突然聊到自己最初開始寫作的原因，因為我是I人，習慣自己找資料、自學，也正因為網路上有這麼多願意無私分享的前輩們，我才能有不斷學習成長的機會，陪我走過一段又一段自學摸索的過程。我一直很感謝「那些人」，也因此，我想成為「那個能用文字幫助別人的人」，帶著這樣初心持續寫文。這篇我想記錄2025年的回顧，並寫下對2026年的期許。

2025讓我成長的事

人工試管療程

這是一段特別的人生體驗，從今年決定進入人工試管療程開始，每一個環節都是身體與心理的雙重挑戰。幸運的是，在這些努力之後，我們成功獲得一顆切片(PGS/PGT-A)後健康的胚胎。

日期 事件 說明
2025/4/22 抽血檢驗 AMH 6.93 是有卵子庫存量的。
2025/5/6 子宮輸卵管攝影(HSG) 這比我想像痛，但我遇到天使，她過程中拍拍我的手，對我說:「呼吸、吐氣，你很棒」。結果輸卵管左邊堵塞、右邊通一個小洞，醫生說這應該就是我難受孕的原因之一。
2025/5/15-5/20 打排卵針第1~6針 開始進入打針與吃藥的例行作業，我是固定在晚上8點左右打針，很感謝這段時間老公幫我打針，自己插針真的需要梁靜茹給勇氣😂。
2025/5/16 抽血檢驗 療程期間需密集回診，大約2-3天就要抽血一次確認數值變化。(連續肚子打針、手臂抽血，這2個部位都瘀青)
2025/5/21 排卵針第7針及抑制排卵針 加入抑制排卵針是為了避免提早排卵。
2025/5/21 抽血檢驗 因為剛好去醫院抽血檢查，所以護士可以順便幫忙打排卵針第7針及抑制排卵針。
2025/5/22 抽血檢驗 監測體內荷爾蒙變化，關鍵項目是E2(雌激素)，同時也會看LH(黃體生成素)、P4(黃體素)等，結合陰道超音波看卵泡大小，來即時調整排卵針劑量。
2025/5/22-5/23 打排卵針第8~9針 隨著針劑累積，腹部會開始有較明顯的脹感。平常月經週期只排一顆卵，現在是同時刺激多顆卵泡成長，因此腹部脹感自然會更加明顯。
2025/5/24 排卵針第10針及抑制排卵針 加入抑制排卵針是為了避免提早排卵。
2025/5/24 抽血檢驗 因為剛好去醫院抽血檢查，所以護士可以順便幫忙打排卵針第10針及抑制排卵針。
2025/5/25 排卵針第11針 排卵針針數是看濾泡生長狀況決定的，若有潛力長大的濾泡，就有可能繼續施打，刺激濾泡成長。
2025/5/26 抽血檢驗 術前最後確認，醫生評估卵泡成熟度，準備取卵。
2025/5/27 破卵針 最後關鍵的針，需在12:15am打一次針，及過2個小時後2:15am再打一次針。
2025/5/28 全身麻醉的取卵手術 隔天我竟然還是去上班，腹部會不舒服、脹痛、需要慢慢走路，但因為我的工作可以坐著，所以可以撐到下班。
2025/6/16 切片報告出爐 通過切片檢驗，確認有1顆是健康胚胎。但因為子宮肌瘤的原因，醫生建議先進行傳統開腹式子宮肌瘤切除手術，手術後需休養3-6個月，再進行胚胎植入。

(我的第一針，護士握著我的手，溫柔但堅定地打下去。)

寫作的斷捨離

今年我發起自我挑戰，嘗試每天產出一篇貼文。在堅持89天後，我按下暫停鍵。不是放棄，而是我意識到為了追求「每天發文」，反而壓縮了撰寫優質深度內容的時間。於是我決定先停下，重新調整節奏，我仍處於寫作的摸索期，我想給自己空間試錯、沉澱，在嘗試中找到能走得更遠的步調。

總結這一年：

  • 產出了 34篇長文章。
  • 累計 91篇貼文。
  • 同步經營 Substack，感謝目前有 6位訂閱者 的早期支持。

英文學習132天

由於今年7月想說去報考 TOEIC，卻一直很難養成固定時間唸英文的習慣，直到我發現了「遊戲化學習」的魔力，結果現在就是每天 EPOP、情勒鳥，持續了 132 天，這是我人生中持續最長的英文自學紀錄。

  • EPOP 體驗: 我第一個使用的英文學習 APP 是 EPOP，有7天免費試用期。我想如果我真的有7天持續打開練習的話，我再考慮年訂閱。而因為 EPOP 把學習變成遊戲化，結果我竟然每天都想打開來使用！
  • 加入情勒鳥: 用了 EPOP 後，我就想說用看看傳說中的「情勒鳥」(Duolingo)。各種經典情勒台詞輪番上陣，例如:「沒關係啊，今天不學沒關係，你就永遠不要學」、「理智地講，你的47連勝還是堅持下去的好，不過，熊言逆耳，你愛聽不見。」，除了情勒，也有給出滿滿情緒價值的「這是人類的速度嗎？一題未錯！承認吧，你的真身其實是超級電腦！」。看到情勒鳥這麼努力幫我維持連勝紀錄，我腦波弱都不好意思不打開來學了，也就真的養成了每天學習英文的習慣。

Python學習51天

因為養成每天學習英文的習慣後，我想是不是程式學習也有類似的APP可以用呢？我驚喜地發現 Brilliant 平台有很多程式及數學的課程，我先從「Thinking in Python」開始，也連續超過51天了!目前還沒課金升級，僅用免費版每天也可以看1-2個單元。

11/30考TOEIC

距離上一次考 TOEIC 已經是2016年，當時的成績是730分。今年莫名地感覺自信心爆棚(情勒鳥給我的😂)，而成績也確實進步了，拿到785分，學英文是一場長期的無限遊戲，但偶爾用 TOEIC 這種有限遊戲來檢視進度、順便給自己一點成就感，其實也滿好的。

ps.分享這次我到台科大考場看到的烏龜群。

內部競賽獲獎

公司今年舉辦 GitHub Copilot 競賽，第一名從缺，而我獲得了第二名🤩。我以「Github Copilot 應用於 ERP SQL 開發」主題參賽，分享怎麼把 GitHub Copilot 實際運用在日常開發與問題解決，透過準備競賽過程中，我也更系統性地整理自己使用 GitHub Copilot 的方式，對工具的應用能力有明顯提升。

2026我想完成的事

子宮肌瘤手術

面對即將到來的2026年，我做了一個決定，因為需要好好處理子宮肌瘤的手術，我決定先離職。這聽起來或許有些冒險，但我視為人生的「主動應劫」，既然身體需要修復，那就乾脆讓自己徹底停下來，把這段時間當作是「休養」與「充電」的契機。

持續內容創作

我熱愛分享學習與實作的過程，對於那些跟我一樣習慣「獨自升級」的戰友們，我希望這些紀錄能成為一份遊戲攻略，讓大家在練等路上預先知道哪條路有寶藏、哪條路有陷阱。

2026年我應該會聚焦在「技術應用」、「落地實作」與「嘗試變現」等主題:

  • 持續學習並實作: 將原本複雜技術概念，轉化為易理解、能實際落地的應用範例。
  • AI 協作開發: 延續去年，分享更多用 AI 輔助寫程式的實作經驗。
  • 養成接案硬實力: 專攻 LINE 機器人串接與 Wordpress 外掛開發。
  • 產品從0到1: 完整紀錄 Spec Coding APP 的開發與上架流程。(完成上架App夢想)

探索服務模式

  • 啟動 Fiverr 接案: 除了開發收入來源，也是為了持續累積實戰經驗，並能接軌國際市場。
  • 啟動1對1諮詢服務: 我也在思考變現與價值的平衡。以前總覺得要寫付費電子書，但心裡總覺得卡卡的，因為既然我的初衷是「分享學習過程」，那把它放在付費內容似乎違背了本心。後來我轉念想:「執行才是最難的」，可能會有人看完文章後，在實際操作時會卡關，或者遇到無法處理的個別問題。因此，我計畫嘗試「陪跑」的模式，啟動1對1諮詢服務，我來幫你看卡在哪裡，甚至手把手一步一步教學，能直接幫助到需要的人，也更符合我寫作的初衷。

繼續英文學習

繼續被「情勒鳥」情緒勒索，維持連勝紀錄。計畫在2026年底再去考一次 TOEIC，希望能突破2025年的成績，拿到金色(860-990)等級證書。

繼續程式學習

除了繼續在 Brilliant 平台學習，計畫在2026年底前完成「Programming & CS」系列課程。並安排時間跟上 Terry 與 Hank 的職涯護城河實戰營 課程，養成 Leetcode 解題腦，逐步提升程式硬實力。

養成運動習慣

運動一直在待辦清單裡，斷斷續續地執行，還沒找到養成固定時間運動習慣的方式。計畫在2026年術後先以溫和的方式開始，像是每天散步30分鐘、睡前做30分鐘瑜伽等，慢慢讓身體適應運動的節奏，然後再逐步增加強度與頻率。

總結

這是我第一次寫年度回顧，寫了才發現這個過程很有意義，也是一次與自己的深度對話，因為會讓我好好思考過去一年裡發生了什麼、學到了什麼，也讓我有機會重新審視自己的目標與方向。

2025年對我來說，從身體試煉到技能持續精進，我發現自己正一步步實踐著「獨自升級」的過程，謝謝在針孔與瘀青中依然勇敢的自己，也謝謝在繁忙中仍持續學習並寫作的自己。而2026年，寫作還是我最想持續做的事，因為我始終記得，自己也是成長於那些願意默默分享的人，一篇文章、一段經驗，慢慢陪著走過來的。我將繼續帶著這樣的初心持續寫文，讓自己成為「那個能用文字幫助別人的人」。

我們明年見!👋

(祝大家都到達心中夢想的「那個地方」。)
