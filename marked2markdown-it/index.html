Markdown 渲染器從 marked 換成 markdown-it

Posted on 2024-05-21 In Hexo

本文說明為何將 Hexo Markdown 預設的渲染器 marked，換成 markdown-it。並詳細列出更多在 Hexo 上的進階 Markdown 用法，如插入emoji、代辦事項、數學公式、螢光筆效果、自定義提示區塊、NexT 預設提示區塊，以及 Mermaid 圖表視覺化。

更換成 markdown-it 原因

hexo-renderer-marked 是 Hexo 預設的 Markdown 的渲染器。而換成 hexo-renderer-markdown-it 主要是他能支援更多 Markdown 格式，最重要的就是能用 LaTeX 語法呈現數學公式。

安裝 markdown-it

在安裝 hexo-renderer-markdown-it 前，要先解除安裝 hexo-renderer-marked。

修改 Hexo config

進入專案根目錄資料夾底下的 _config.yml，找到 # Deployment 區域，通常在最下面。

是專案根目錄資料夾底下的 config 檔
(不是 themes/next 資料夾底下的 config 檔)

將 # Markdown 貼在 # Deployment 區域上方。

在看網路很多資料時，發現 hexo-renderer-markdown-it 是可能會造成 TOC (Table of Contents) 壞掉的，我也有遇到這個問題，就是點擊文章目錄不會自動跳轉到該文章地方，我最後解決的方式是調整 anchors: 的參數就可以了。

進階 Markdown 用法 (持續更新)

數學公式

MathJax 是 JavaScript 函式庫，可以用來以 Latex 文法來寫數學公式。

$$ e^{ \pm i\theta } = \cos \theta \pm i\sin \theta $$

確認已安裝 markdown-it

啟用 MathJax 前，需先確定已完成安裝 hexo-renderer-markdown-it。

啟用 MathJax

進入 themes/next/_config.yml 中，找到 math:，啟用 mathjax。

更多 NexT 主題中所支援的數學公式說明請看 NexT 主題官方文件-Math Equations。

安裝 markdown-it-container

修改 Hexo config

進入專案根目錄資料夾底下的 _config.yml，找到 # Markdown 區域的 plugins:。如果你這段本來就有其他套件，那可以直接加在其他套件下方就好。我這邊自定義了 2 個提示區塊，tip 和 fact。

自定義CSS樣式

📍 啟用 styles.styl

進入 themes/next/_config.yml 中，找到 custom_file_path:，啟用 style: source/_data/styles.styl，位置在 32 行。

📍 新增並編輯 source/_data/styles.styl 檔案

是專案根目錄資料夾底下的 source 資料夾
(不是 themes/next 資料夾底下的 source 資料夾)

啟用 Note

  • icons: 改為 true。
  • style: 改為 flat。

其他提示區塊說明請看 在hexo-NexT中插入note提示块。

Mermaid 圖表

Mermaid 是一種用於描述圖表的文本語言，支持多種圖表類型如流程圖、時序圖、甘特圖、類別圖(UML)、Git版控圖、使用者旅程圖、象限圖、XY 圖表、圓餅圖等。

網頁呈現樣子:


sequenceDiagram
participant Alice
participant Bob
Alice->>Bob: Hi Bob
Bob->>Alice: Hi Alice

網頁呈現樣子:


pie
"Apples": 1050
"Bananas": 890
"Oranges": 740

安裝 hexo-filter-mermaid-diagrams

啟用 mermaid

進入 themes/next/_config.yml 中，找到 mermaid:，並啟用。

更多 Mermaid 所支援的圖表請看 Mermaid 官方文件。
