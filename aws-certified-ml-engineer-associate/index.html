Pass AWS Certified Machine Learning Engineer – Associate

Posted on 2024-12-27 In AWS

一次考上 AWS MLA 證照，本篇記錄下參與 AWS Certified Machine Learning Engineer – Associate 認證考試的過程，並整理相關資源，希望能對想要挑戰這張證照的朋友們有所幫助！

為什麼想考?

暨 11 月考完 人生中第一張 AWS 證照 - AWS Certified AI Practitioner 後(考上癮)，又看到有 Get AWS Certified: Associate 挑戰賽 的 50% 考試優惠折扣，而且還可得到 AWS AI Ready 大禮包 +1。(我想要有2個這個酷東西 - I am AI-ready T-shirt)

這張證照也會有數位徽章，是透過 Credly 的平台，讓你可直接加入 LinkedIn 的喔!

考試資訊

  • 考試時間: 130 分鐘
  • 考試形式: 65 個問題
  • 費用: 150 USD
  • 測驗選項: Pearson VUE 測驗中心或有監考人員的線上考試
  • 提供的語言: 英文、日文、韓文和簡體中文

前往 AWS 官方查看 AWS Certified Machine Learning Engineer – Associate 考試資訊。

考試成績

100–1000 分，最低通過分數為 720 分。

問題類型

  • 單選題: 1 個正確答案及 3 個錯誤答案。
  • 多選題: 從 5 個或更多選項中包含 2 個或以上正確答案。需選擇所有正確答案。
  • 排序題：提供 3–5 個選項，需按照指定任務選擇正確答案並以正確順序排列。
  • 配對題：提供一組選項與 3–7 個提示，需將選項與提示正確配對。
  • 案例題：基於單一案例提出 2 個或更多問題。

考試範圍

  • ML 資料處理 (28%)
  • ML 模型開發 (26%)
  • ML 工作流程的部署與配置 (22%)
  • ML 的監控、維護與安全性 (24%)

前往 AWS 官方查看 AWS Certified Machine Learning Engineer – Associate 考試指南。

怎麼準備考試?

AWS官方課程

  1. Exam Prep Standard Course: AWS Certified Machine Learning Engineer - Associate (MLA-C01 - Simplified Chinese)
  2. Exam Prep Standard Course: AWS Certified Machine Learning Engineer - Associate (MLA-C01 - English)
  3. Standard Exam Prep Plan: AWS Certified Machine Learning - Associate (MLA-C01)

免費考試題庫

  1. Exam Prep Official Practice Question Set: AWS Certified Machine Learning Engineer - Associate (MLA-C01 - Simplified Chinese)
  2. Exam Prep Official Practice Question Set: AWS Certified Machine Learning Engineer - Associate (MLA-C01 - English)
  3. 非官方85題-ExamTopics
  4. 我整理的85題-考題和心智圖

總結

這次通過 AWS Certified Machine Learning Engineer – Associate(MLA-C01) 的認證，分數比之前考取的 AWS Certified AI Practitioner(AIF-C01) 還要高，因為兩張證照之間有許多重疊的核心觀念。AI Practitioner 是基礎，專注於人工智慧的基本概念與 AWS 相關服務的入門知識，而 ML Associate 則進一步深入到了機器學習的具體應用，包括數據準備、模型開發、部署與監控。

我想考試不僅是對技術知識的驗證，更是一種激勵自我延伸學習的方法，希望這篇經驗分享能夠幫助更多想要挑戰 MLA-C01 認證的朋友們，讓我們一起踏上學習與成長的旅途吧！
