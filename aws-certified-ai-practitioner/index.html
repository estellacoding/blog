人生中第一張 AWS 證照 - AWS Certified AI Practitioner

Posted on 2024-11-21 Edited on 2024-12-25 In AWS

我一次考上人生中第一張 AWS 證照、也是人生第一張 AI 證照！記錄下參與 AWS Certified AI Practitioner 認證考試的過程，並整理相關資源，希望能對想要挑戰這張證照的朋友們有所幫助！

為什麼想考?

因為剛好發現 AWS AI 人才就緒計畫 的活動，加入認證加速營就可以得到 「USD50 考試優惠價 + 免費重考」的機會，而且前 300 名考取此認證且為首次取得 AWS 認證者，還可得到 AWS AI Ready 大禮包。(我想要這個酷東西 - I am AI-ready T-shirt)

當然除了上述外，這張證照會有數位徽章，是透過 Credly 的平台，讓你可直接加入 LinkedIn 的喔!

考試資訊

  • 考試時間: 90 分鐘
  • 考試形式: 65 個問題
  • 費用: 100 USD
  • 測驗選項: Pearson VUE 測驗中心或有監考人員的線上考試
  • 提供的語言: 英文、日文、韓文、萄葡牙文 (巴西) 和簡體中文

前往 AWS 官方查看 AWS Certified AI Practitioner 考試資訊。

考試成績

100–1000 分，最低通過分數為 700 分。

問題類型

  • 單選題: 1 個正確答案及 3 個錯誤答案。
  • 多選題: 從 5 個或更多選項中包含 2 個或以上正確答案。需選擇所有正確答案。
  • 排序題: 提供 3–5 個選項，需按照指定任務選擇正確答案並以正確順序排列。
  • 配對題: 提供一組選項與 3–7 個提示，需將選項與提示正確配對。
  • 案例題: 基於單一案例提出 2 個或更多問題。

考試範圍

  • AI 和 ML 的基礎知識 (占20%)
  • 生成式 AI 的基礎知識 (占24%)
  • 基礎模型(FM)的應用 (占28%)
  • 負責任 AI(Responsible AI)的行為守則 (占14%)
  • AI 解決方案的安全性、合規性及治理 (占14%)

前往 AWS 官方查看 AWS Certified AI Practitioner 考試指南。

報名注意事項

考試語言

我推薦可以報名「簡體中文版」喔，因為考試時每題都可以看英文原版的題目內容，所以就不需要擔心簡體中文翻譯會看不懂的問題。

線上考試

我是報名線上考試!(線上考試也無法作弊😂)

考試前，要先用參加考試的同一台電腦、網路連線和房間執行系統測試，系統測試會確認電腦聲音、鏡頭都可正常運行，也需要關閉電腦中所有的應用程式，包含瀏覽器。建議用個人電腦，因為公司電腦可能會有防火牆或安全設定導致系統無法通過測試。

考試當天，我是報名早上8點，7點半就要進入考試環境，一進入就都是打開攝影機的狀態，跟著系統測試再完成相關流程，並且會要你用手機拍攝電腦前後左右和身份證的照片，接著還會出現監考人員用英文說話(突然出現聲音讓我本來半睡半醒都嚇醒了哈哈)，請你再用電腦鏡頭看一次你的環境四周，我當時手機放在桌上，他有叫我把手機放遠到其他地方。而整個考試期間，應該都是在遠端監控你的環境。

當天要記得拿身分證(或是護照或是駕照)。

線上考試政策:

  • 所有個人物品必須放置在無法取得的地方，且在任何時間不得接觸或使用。
  • 個人物品包括但不限於電子設備(如手機、手錶、攝影機或其他錄影設備)、包包、公事包、錢包、書籍及筆記。
  • 不允許離開座位，包括上廁所或任何其他類型的休息。
  • 不允許使用草稿紙或做筆記。
  • 不允許自言自語或出聲說話，除非與監考人員進行溝通。
  • 不允許吃東西、喝飲料或使用個人清潔用品，因為這些物品不得放置在可接觸的範圍內。

相關報名考試步驟細節，請看 AWS認證報名考試詳細教學。

怎麼準備考試?

時間有限的情況下，我採用了懶人備考策略:

  • 先把 AWS 官方題庫 和 非官方題庫 認真看完。
  • 看完題庫，再遮住答案確認自己有答對。
  • 答錯的地方記錄到筆記中。
  • 有時間再確認所有題目的選項是理解清楚的。

總結

如果人生一定會考 AWS 證照的話，建議可以從 Practitioner(基礎級) 這個類別開始考。

  • AWS Certified AI Practitioner
  • AWS Certified Cloud Practitioner

因為只要通過任一次認證就可以取得考試半價券，考試半價券效期有3年，所以對於計畫中還想繼續報考其他證照，像是 Specialty(專家級)原價 300 USD 有半價的話，就可以省超多的喔。
